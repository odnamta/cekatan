# Landing Page Polish Roadmap

> Items that require real assets, external services, or significant content work.
> These cannot be tackled with code alone — each needs business/design input.

## Priority 1: Content Assets

- [ ] **Real product screenshots** — Replace abstract SVG mockups with actual dashboard/assessment screenshots (requires production data)
- [ ] **Customer testimonials** — Collect 3-5 quotes with photos and org names from GIS/GLS users
- [ ] **Video demo** — 60-90 second product walkthrough for hero or feature section

## Priority 2: Design & Branding

- [ ] **Custom illustrations** — Commission branded illustrations (onboarding flow, empty states, error pages)
- [ ] **Favicon + app icons** — Replace basic SVG with professional multi-size icon set
- [ ] **Social preview images** — Per-page OG images (assessment results, cert verification)

## Priority 3: Business Sections

- [ ] **Pricing section** — Define tiers (Free / Pro / Enterprise) with feature matrix
- [ ] **Blog / Resources** — Content marketing section for SEO (assessment best practices, guides)
- [ ] **Case study page** — Detailed GIS implementation story with metrics

## Priority 4: Analytics & Optimization

- [ ] **Vercel Analytics** — Enable Web Vitals + audience tracking
- [ ] **Heatmaps** — Hotjar or PostHog for scroll depth and click tracking
- [ ] **A/B testing** — Test hero copy variants, CTA colors, section order
- [ ] **Performance audit** — Lighthouse CI, bundle size optimization

## Priority 5: Trust & Compliance

- [ ] **Privacy policy page** — GDPR/Indonesian data protection compliant
- [ ] **Terms of service page** — Proper legal content
- [ ] **Security page** — Detail RLS, encryption, data isolation
